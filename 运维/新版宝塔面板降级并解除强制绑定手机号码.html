新版宝塔面板降级并解除强制绑定手机号码 ​

方法1（降级法） ​

先正常安装宝塔最新版，之后运行以下命令降级到v7.7.0版本，就可以继续使用之前的解除方法了

bash
#下载v7.7.0版本文件
wget http://download.bt.cn/install/update/LinuxPanel-7.7.0.zip

#本站备份v7.7.0文件下载
wget https://oss.tm9uywdl.top/d/Tools/Linux/LinuxPanel-7.7.0.zip?sign=hcem2QWkV3YEEkIFj42ZI-ZciWFXeZKmKARaY9XeVWY=:0
bash
#安装v7.7.0版本命令
unzip LinuxPanel-7.7.0.zip
cd /root/panel
bash update.sh

降级成功后，直接SSH运行以下两条命令删除强制绑定手机页面文件

bash
sed -i "s|if (bind_user == 'True') {|if (bind_user == 'REMOVED') {|g" /www/server/panel/BTPanel/static/js/index.js
rm -rf /www/server/panel/data/bind.pl

方法2（海外版） ​

直接安装宝塔国际版，国际版目前还未强制要求绑定手机（之后估计也不会要求绑定手机），所谓国际版，其实就是英语界面而已，其他没啥区别~ 宝塔国际版地址：https://www.aapanel.com
